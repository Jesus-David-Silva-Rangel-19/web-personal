Monitor Digital de Presión Arterial (MDPA)

Notas del Proyecto

El Monitor Digital de Presión Arterial (MDPA) es una aplicación web en tiempo real para el monitoreo de la presión arterial. La aplicación permite a los usuarios monitorear su presión arterial en tiempo real, ver gráficos interactivos, recibir alertas personalizadas y generar reportes.

El MDPA fue desarrollado por Jesús David Silva Rangel, un Científico De Datos e Ingeniero de Machine Learning de Colombia. El proyecto fue desarrollado utilizando Python, Machine Learning, JavaScript, Flask, Bootstrap, Chart.js, HTML y CSS.

El MDPA es un proyecto innovador que utiliza tecnologías modernas para ayudar a las personas a monitorear su presión arterial de forma segura y eficaz.
